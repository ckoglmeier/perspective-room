# Changelog

## v0.1.0

Initial release of Perspective Room Standard.

Includes:

- `perspective_room.v1` schema
- TypeScript types for rooms, claims, materials, trust state, and bundles
- Validation helper for required fields and trust-state warnings
- Bundle renderer for `room.json`, `materials.json`, `agent.md`, and `index.html`
- Minimal example bundle
- Full seed-room example bundle
- Agent-readiness eval fixtures
