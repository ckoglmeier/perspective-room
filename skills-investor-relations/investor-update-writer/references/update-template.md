```text
Subject: [Company] investor update - [Month YYYY]

- [Specific win/change]

- [Metric]: [value] ([delta/context])

Asks
- [Specific intro/help/request]

Risks / follow-ups
- [Risk or follow-up with owner/next step if available]

```

## Rules

- Prefer concrete asks over generic "help appreciated."
- Include misses when they matter.
- Do not bury the lead.
- Link to the room only when it has meaningful changes or helps answer diligence.
